# whaticket-multiconexao-baileys
# Whaticket Multiconexão com Baileys  Este projeto adapta o Whaticket Community para suportar múltiplas conexões WhatsApp utilizando a biblioteca Baileys.  Principais funcionalidades: - Suporte a múltiplos números de WhatsApp - Associação por setor/departamento - Envio de mensagem via API escolhendo a conexão - Deploy em Linux
